# Java Operators

This folder contains Java programs demonstrating different types of operators.

## Files Included

- ArithmeticOperators.java
- AssignmentOperators.java
- ComparisonOperators.java
- LogicalOperators.java
- UnaryOperators.java
- TernaryOperator.java

## Topics Covered

### Arithmetic Operators
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modulus (%)

### Assignment Operators
- +=
- -=
- *=
- /=
- %=

### Comparison Operators
- ==
- !=
- >
- <
- >=
- <=

### Logical Operators
- &&
- ||
- !

### Unary Operators
- Increment (++)
- Decrement (--)

### Ternary Operator
- Conditional Operator (? :)

## Purpose

These examples are part of my Java learning journey and cover the fundamentals of Java operators with simple beginner-friendly programs.
